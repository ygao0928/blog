# 工厂模式

# 简介：

工厂模式又叫工厂方法模式，是一种创建设计模式，在父类中提供一个创建对象的方法，允许子类决定实例化对象的类型。

主要为了：定义一个创建对象的接口，让之类自己决定实例化哪一个工厂类，工厂模式使其创建过程延迟到之类进行。
