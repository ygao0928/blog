### 订单评审

==OrderAppraisalController==

获取订单评审

#### 三方评审

> 评审状态:kissing_smiling_eyes: 
>
> 0:待评审;1:评审不通过;2评审通过  

##### 评审操作

地址:`/orderAppraisal/review`

==参数:==

- json

> 参数列表,针对不同的部门有不同参见下面.
>
> 评审信息:
>
> statue:0 代表初建待评审;1:评审不通过;2:评审通过
>
> 订单本身状态:
>
> 0: 初建待评审;1:评审不通过;2:评审通过;3:进入生产

|   参数   |           说明            |
| :------: | :-----------------------: |
|  `flag`  | 评审状态(通过or驳回`int`) |
| `userId` |         操作人id          |
| `remark` |         评审意见          |
| orderId  |          订单id           |

参数:{"flag":2,"userId":4,"orderId":,"remark":"good"}

##### 列表展示
